--- layout: post title: Improve performance of DI Studio client date: '2014-12-16T09:11:00.000-08:00' author: Allan Bowe tags: modified_time: '2015-06-02T05:54:28.811-07:00' thumbnail: /assets/images/DI_MemSize.PNG blogger_id: tag:blogger.com,1999:blog-3488921339026456633.post-8238173319155671561 blogger_orig_url: http://www.rawsas.com/2014/12/improve-performance-of-di-studio-client.html --- You can speed up the performance of your DI Studio client by increasing memory via the JavaArgs_1 variable in the distudio.ini file.  A suggested approach is to increase from -Xmx512m to -Xmx1024m.


This is further documented in usage note 41571.  A similar update can be made to SMC (useful when importing / exporting large packages).

Note that the max memory that will go in JavaArgs is typcially 1536.
