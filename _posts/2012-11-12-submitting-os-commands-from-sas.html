--- layout: post title: Submitting OS commands from SAS date: '2012-11-12T14:27:00.000-08:00' author: Allan Bowe tags: - os command - x command modified_time: '2012-11-25T11:01:26.240-08:00' blogger_id: tag:blogger.com,1999:blog-3488921339026456633.post-6788709141273163331 blogger_orig_url: http://www.rawsas.com/2012/11/submitting-os-commands-from-sas.html --- When dealing with a Pooled Workspace Server, a generic 'host account' is normally used.  So you tend not to have individual host accounts to log in with.  Therefore, sometimes the only way to navigate the server is to use EG!  Hence this piece of code (used to grep files but can be easily modified for other commands)..


%macrogrep(loc,string);
filename x pipe "cd &loc; grep -irl &string .";
data _null_;
infile x;
input;
put _infile_;
run;
%mend;
/* check logs */
%grep(C:\myDirLoc,searchString)
