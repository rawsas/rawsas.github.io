  • The macro needs to be called exactly the same as the filename.
  • Define all local macro variables used within the macro with the %local  statement.
  •  DO NOT USE POSITIONAL PARAMETERS (value1,value2), far better to use keyword parameters (name=value), as this means your macro can be exteneded / modified without breaking existing code.
  • TAB's should be avoided at all costs, as they will render differently in various editors.  The SAS R&D standard is 3 spaces.  Your EG program editor, and notepad++, can be configured to apply this by default when hitting the tab button.
  • Options pagesize=max;
